## 📌 Project Overview

This project focuses on analyzing Uber NCR ride booking data using **SQL** and visualizing key business insights through an **interactive Power BI dashboard**. The objective is to extract meaningful insights from ride booking records and answer real-world business questions related to revenue, customer behavior, vehicle performance, cancellations, payment preferences, and operational efficiency.

The SQL analysis is complemented by a Power BI dashboard that provides interactive visualizations, KPI cards, AI-powered Key Influencers, and dynamic filters, enabling users to explore booking trends, revenue performance, cancellation patterns, and ride analytics across different dimensions.

The analysis demonstrates the practical application of SQL concepts such as:

- Data Filtering
- Aggregations
- GROUP BY Operations
- Common Table Expressions (CTEs)
- Window Functions
- Business KPI Analysis

The Power BI dashboard showcases:

- 📈 Revenue Trends Over Time
- 🚖 Revenue by Vehicle Type
- 📍 Pickup Location Analysis
- ❌ Customer vs Driver Cancellation Comparison
- 📏 Ride Distance Analysis
- ⭐ Average Driver Rating
- 📊 Interactive KPIs and Dynamic Filters
- 🤖 AI-Powered Key Influencers for Booking Completion

## 📷 Dashboard Preview
![Uber Performance Dashboard](Uber%20Performance%20Dashboard%20Image.png)

The **Uber Performance Dashboard** is an interactive Power BI dashboard designed to provide a comprehensive overview of Uber NCR ride booking performance. It transforms raw ride booking data into actionable business insights through dynamic visualizations, enabling stakeholders to monitor operational efficiency, revenue generation, customer behavior, and ride performance.

The dashboard features key performance indicators (KPIs) such as **Average Ride Distance** and **Average Driver Rating**, providing a quick snapshot of overall service quality. Users can interact with the dashboard using filters for **Pickup Locations** and **Ride Distance**, allowing detailed exploration of booking patterns across different locations and ride ranges.

### Dashboard Highlights

- 📈 **Revenue Analysis**
  - Revenue generated by each vehicle type.
  - Monthly revenue trend analysis to identify peak earning periods.

- 🚖 **Vehicle Performance**
  - Comparison of revenue contribution across vehicle categories including Auto, Go Mini, Go Sedan, Bike, Premier Sedan, eBike, and Uber XL.

- ❌ **Cancellation Analysis**
  - Doughnut chart comparing customer and driver cancellation rates to understand operational challenges.

- 📍 **Location-Based Analysis**
  - Interactive pickup location slicer for analyzing booking performance across various NCR locations.

- 📏 **Ride Distance Analysis**
  - Dynamic ride distance filter to evaluate booking trends based on travel distance.

- 🤖 **AI-Powered Key Influencers**
  - Power BI's **Key Influencers** visual identifies the factors that increase the likelihood of a booking being completed, helping uncover important business drivers without manual analysis.

- 📅 **Revenue Trend Analysis**
  - Monthly line chart showing revenue fluctuations throughout the year to identify seasonal demand patterns.

### Interactive Features

- Pickup Location Filter
- Ride Distance Slider
- Dynamic Visual Cross-Filtering
- AI-Based Key Influencers Visual
- Responsive KPI Cards
